# MP34DT01
Arduino library to support MP32DT01 digital omnidirectionnal microphones

## Note

This library uses the DFSDM module. Only boards with DFSDM module are compatible.

## API

The library provides the following API:

* begin()
* end()
* record()
* stop_recording()
* pause_recording()
* resume_recording()
* set_volume()
